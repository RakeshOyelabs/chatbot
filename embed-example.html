Third-Party Website Example

Welcome!

This is an example of how to embed the oyelabs ai chatbot into your website using an iframe.

The chatbot will appear in the bottom right corner of your screen. Click the green button to start chatting!

Integration Instructions

To embed this chatbot in your website, add the following code to your HTML:

<!-- Add this before closing </body> tag -->
<iframe
  src="http://localhost:5173?embed=true"
  style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; pointer-events: none; z-index: 9999;"
  scrolling="no"
></iframe>

<script>
  // Allow pointer events only on the chatbot area
  const iframe = document.querySelector('iframe');
  iframe.style.pointerEvents = 'none';
  iframe.contentWindow.document.body.style.pointerEvents = 'auto';
</script>
