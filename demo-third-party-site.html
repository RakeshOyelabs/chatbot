This is a Third-Party Website Demo

This page demonstrates how the chatbot widget appears when embedded in a third-party website using an iframe.

Notice: The chatbot widget in the bottom-right corner is loaded via iframe and works independently of this page's content.

About This Demo

This is a standalone HTML page that simulates a third-party website. The chatbot widget you see in the bottom-right corner is embedded using an iframe and will work on any website.

How It Works

The chatbot is embedded using a simple iframe tag that loads the chatbot application from a separate source. This means:

  • The chatbot runs in an isolated environment
  • It doesn't interfere with your website's JavaScript or CSS
  • It can be added to any website with just a few lines of code
  • It maintains its own state and functionality

Integration

To add this chatbot to your website, you just need to include the iframe code in your HTML. Check the IFRAME_EMBED_GUIDE.md file for complete instructions.
